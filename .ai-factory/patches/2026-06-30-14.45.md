# Patch 2026-06-30 14:45

## Context

Token and usage totals were formatted inconsistently across run-related UI.
The selected Flow node header rendered raw integers, several run/task/studio
surfaces used fixed `en-US` formatting, and the right run inspector rail reused
those values during live refreshes.

## Changes

- Passed the active `next-intl` locale into run detail, scratch-run detail,
  task detail, Observatory, and Studio assistant usage surfaces.
- Formatted selected-node token totals, per-attempt token totals, inspector rail
  cost facts, node token labels, task aggregates, task run history, timeline
  token totals, Observatory cost tables, and Studio assistant usage meters with
  locale-aware `Intl.NumberFormat(locale)`.
- Kept the Flow editor assistant sweep scoped to visible usage counters; no
  token/cost totals are rendered directly by the Flow editor action result.
- Added regression coverage for RU-style grouped token totals in the shared cost
  facts builder, Flow run center, and timeline.
