**Source**: 2026-05-26-12.45.md, 2026-05-26-12.53.md
**Rule**: When fixing a single Zod field whose value flows into `path.resolve` / `path.join` / `fs.*` / `child_process.spawn` argv / SQL / HTML, audit every sibling field in the same schema in the same pass.
